# Julie Fonseca — Autism Parent Mentor

Bilingual landing page (Spanish primary, English secondary).

Built for iLaunch U Marketing — static HTML/CSS/JS deployed to Vercel.

## Local preview

```bash
python3 -m http.server 4324
```

Then open http://localhost:4324

## Stack
- Single-file `index.html`
- Google Fonts (Playfair Display + Inter)
- Hero images generated with Nano Banana Pro (gemini-3-pro-image-preview)
- Deployed: Vercel (auto-deploy from `main`)
